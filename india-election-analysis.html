You can download the dataset along with the questions from this link.

Data

There are two tables, constituency_wise_results2014.csv and constituency_wise_results2019.csv. As the names speak for themselves, both the tables contain the data of how many votes each candidate contesting in each constituency in every state got and the number of electors present in each constituency. The columns include pc_name (constituency), state, candidate, candidate information (sex, age, category), party and party symbol (the candidate belongs to), general_votes, postal_votes, total_votes and total_electors.

Data Cleaning

The data looks all good in the first look, but upon close inspection, I identified some inconsistencies and cleaned the data in jupyter notebook using python. The file is available in my github.
1. Trailing spaces in pc_name column values. Trailing Space means an empty space character at the ending of a string. Suppose, I want to see the details of 'Adilabad' and I write an appropriate sql query or python command, I will get empty table or dataframe. The reason is that the 'Adilabad' is stored as 'Adilabad ' in the table. Use the below code to strip away the end spaces. 


for col in cols:
data14[col] = data14[col].astype('str')
data14[col] = data14[col].str.strip()

2. Odisha, Chattisgarh and Telangana states are missing in 2014 data. Get the constituencies of these states from 2019 data and change the values in 'state' column (for corresponding constituencies) to the respective states in 2014 data. 


pcs_odisha = data19.loc[data19['state']=='Odisha','pc_name'].unique()
for constituency in pcs_odisha:
    data14.loc[data14['pc_name']==constituency,'state'] = 'Odisha'

3. Three constituencies ('Hamirpur' (in HP & UP) 'Aurangabad' (in Bihar & Maharasstra), 'Maharajganj' (in Bihar & UP)) exist with same name in two different states. Add a prefix in constituency name to make them distinct. 


data14.loc[(data14['state']=='Himachal Pradesh') & (data14['pc_name']=='hamirpur'),'pc_name'] = 'hamirpur_HP'
data14.loc[(data14['state']=='Uttar Pradesh') & (data14['pc_name']=='hamirpur'),'pc_name'] = 'hamirpur_UP'
